微软：由于错误的研发方向，Windows Vista在历经两年多的研发后，全部推翻重写。



微软在经历XP的成功之后，决心要再为科技界搞出一个大新闻。2002年左右立项Windows Vista。当时提出的目标是：

1、支持新语言C#，所有的操作系统软件都改用C#来写，C#运行慢，但开发快。（想干掉开源Linux操作系统）
2、开发Windows File System，将每一个文件存成一个数据库。
（想干掉Oracle、IBM的数据库，同时提防Google抓取这些数据）
3、开发Avalon，让用户在浏览器中得到和传统应用软件相同的体验。
（想开创全新的应用使用方式）


从这些目标所涵盖的方面和所针对的对手，就可以看出当时微软的心有多大了，倘若成功了，未来也许就真的不一样了。

可悲哀的是，理想太丰满，现实太骨感，并没有舞台能够装得下微软这颗心。C#太慢，操作系统根本走不起来；数据库太慢，做出来的档案系统，还不如去档案馆一本一本翻来得快。最最致命的是可预见短期内并没有处理器能够带得动这个系统。

在信仰和情怀的支撑下，微软人坚持折腾了两年............可依旧看不到未来。2004年秋天，微软决定..........全部推翻重做！！！
悲壮！！！！
C#不要了，Windows File System不要了，Avalon能留多少留多少。

2005年7月22日，微软宣布 Windows Vista 为这款新操作系统的名字。微软于2006 年11月2日完成GA版本，向OEM 和企业用户发布。2007年1月30日，正式向普通用户出售，这是Windows 历史上间隔时间最久的一次发布。

Windows Vista经历过推翻重写，间隔了最长时间发布。却熬不过前辈XP，也斗不过后生win7，犹如过眼云烟，不被大众追捧。可程序就是在试错中完善，终究在前进，走点弯路又何妨？
