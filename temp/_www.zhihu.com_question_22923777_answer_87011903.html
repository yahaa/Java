一. 在国内，某项目快结束前，日本那边派了个人过来验收。
1. 我们每天照旧19点、20点下班，这位天天22点下班。
2. 某个周五我看他脸色通红，头晕目眩就要挂了的样子。周末居然没来加班。。周一才来上班。才知道，他病重回国看了病又来了。
3. 验收预定某周六结束，这位周日中午12点的飞机。周六大家通宵加班，我趴在台子上玩了半个晚上的游戏，搭椅子睡了半个晚上。这位一通宵没睡，眼睛红的像兔子，一直干到周日中午10点才离开。

二. 在岛国某项目A
1. 本人工作激情高昂，电脑前一坐就是4个小时。岛国同事劝我，现在是休息时间，一定要休息。“休息”两个字知道不？跟着我念：“休憩きゅうけい”。
2. 一开始的2周，每天下班前，岛国同事都特意过来和我说：嗯，你可以下班了，我们还要继续加班的，你们回去吧，明天再干活。等定休日再一起出去玩。
3. 说好周五下班开车去山梨县泡温泉，碰巧岛国同事那天纳品，我说就算了下次吧。结果当天晚上他们23点纳品结束，凌晨2点来接我开车去泡温泉，感动。

三. 在岛国某项目B
1. 项目开始前有段学习期，就是旧的设备摆弄摆弄，照着操作手册玩一遍。上班在山上，住在山脚下，每天上下班只有一班巴士。每天下班就我和另外2个中国同事按时坐巴士下班，其他一样处于学习期的日本人都是天天22点打车回山下。
2. 我和其他日本人工作量差不多，自己预估2周的工作量。汇总计划会议的时候，前面几个日本人开始报计划，“2个月吧”，"2.5个月吧"，“1个月+2.5个月吧”，擦，轮到我了，赶紧把2周改成3个月。
3. 项目经理(日本人)过来问加班计划。日本同事纷纷表示除了病假一周七天努力加班，我的2个中国同事也表示周六要意思意思。然后到我了：
=> 经理：“嗯，你的进度晚了，这周六周日加班吧？”
=> 我：“周五前我可以挽回进度，不加班”
=> 经理：“那下周进度还会晚的吧，下周六加班吧？”
=> 我：“进度不晚，不加班”
=> 经理：“你的工作量不够，分担点其他工作，加班吧？”
=> 我：“我的工作量已经排满了，加不进来了。”
=> 经理：“#￥%……”
第二天我就被投诉了。。。
4. 坐在我后面的日本妹纸，每天上班白天就双手抱胸，对着屏幕偶尔点点头，屏幕上的内容3天没更换过，真不知道一天天这样怎么熬得住？傍晚吃好晚饭，伸个懒腰，才开始干活。
5. 换座位，抽屉里有一张岛国同事落下工资单，看了下，唉。。。他们是真心不加班没法活啊。

-----------------------------------------------------
大家好热情，补充一些。
1. 日本很多公司有定时退社日，这个日子大家都是不加班的，我上面写的2家公司碰巧都是每周三和每个月发工资的日子是定时退社日。每周总得加班时间也是有上限的，超过还要额外申请。
2. 只有一线人员有加班费，工资+加班费+奖金总额甚至要高过管理人员的薪水。管理层的经理、部长他们都是没有加班费的，往往这些人是60,70年代出生的经历过日本80年代的黄金十年，他们有着极高的工作热情，往往身先士卒天天很晚回去，上班时间也是努力工作不打酱油。
3. 项目B里我也有加班费而且不低，但是我就不喜欢为了磨洋工而加班。
4. 关于被投诉，我在该项目里属于key-man，这个150人的项目，实质上是靠10人左右的技术核心撑起来的。因此即便被投诉对我也没什么影响。（当然，那个坐我后面天天发呆的mm也被投诉了，后来就被换走了。。。）
